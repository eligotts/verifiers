# AGENTS.md

This guide covers best practices for building environments with `verifiers` and using them for evaluation and training. 

The `verifiers` repository is located [here](https://github.com/PrimeIntellect-ai/verifiers). For guidance on contributing to the core `verifiers` library, see [verifiers/AGENTS.md](https://github.com/PrimeIntellect-ai/verifiers/blob/main/verifiers/AGENTS.md).

Full documentation for the `verifiers` library is available [here](https://docs.primeintellect.ai/verifiers/overview).
